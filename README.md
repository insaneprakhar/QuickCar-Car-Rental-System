# QuickCar-Car-Rental-System
A web-based car rental system developed using PHP, MySQL, HTML, CSS and JavaScript.

# **Features**:

1)User Registration and Login

2)Car Listing

3)Car Details Page

4)Car Booking System

5)Booking History

6)Admin Panel

7)Responsive User Interface

# Technologies Used
PHP
MySQL
HTML
CSS
JavaScript
XAMPP
